# Unbinding Kinetics Data

Supporting materials for the Unbinding Kinetics preprint ([Smith et al., ChemRxiv. 2025](https://chemrxiv.org/engage/chemrxiv/article-details/67ab714e81d2151a02fa2c3d)). We provide the input Maestro (.mae) files for each system investigated in the main text. Files contain a single receptor-ligand pair, use the naming convention `{receptor}-{ligand}`, and are organized into directories for each receptor:

- `p38`: p38 Mitogen-Activated Protein Kinase
- `CDK8`: Cyclin Dependent Kinase 8
- `A2A`: Adenosine A<sub>2A</sub> Receptor
- `T4L`: T4-type Lysozyme
- `Trypsin`: Beta Trypsin
